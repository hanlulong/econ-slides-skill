\scriptopening{0.4 min}{%
  This opening states the comparison and its limit. The assignment rule is
  useful but may still select places on different paths. I will show the
  timing, the diagnostic evidence, and the post-period pattern. By the end,
  the audience can separate the reported association from the causal claim
  that the present design does not yet establish.}
\scriptframe{Rollout timing identifies the effect}{0.5 min}{%
  Start with assignment. Firms do not choose the connection year assigned to
  their town. That helps, but a predetermined score may still favor places
  with stronger projected growth. The useful comparison is therefore treated
  firms against firms not yet treated, with the score itself visible rather
  than hidden. The next diagnostic determines whether that comparison is
  credible enough for a causal interpretation.}
\scriptframe{Effects build over three years}{0.6 min}{%
  Read the time axis first. The initial view isolates coefficients before
  connection. \Click{2} The next view marks arrival. \Click{3} The last view
  adds later years. The post-period pattern grows gradually, but that shape
  cannot erase a pre-period difference. We therefore describe the estimates
  as an association and state which new comparison would be needed to go
  further.}
